# Devansh Vishwakarma Portfolio

Plain HTML, CSS, JavaScript, and JSON portfolio site.

## Files

- `index.html` contains the page structure.
- `styles.css` contains all styling and responsive layout rules.
- `script.js` loads content, renders sections, and handles interactions.
- `data.json` stores portfolio content so text, links, skills, projects, and education can be edited without touching markup.
- `public/images` and `public/fonts` contain the static assets.

## Run locally

```bash
npm install
npm run dev
```

The site runs at `http://127.0.0.1:3000/`.

## Build

```bash
npm run build
```

The production files are generated in `dist/`.
